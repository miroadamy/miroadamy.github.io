The general idea

1
2
3
4
5
6

#!/bin/bash
# Parameter: tag or sha1
git checkout --orphan temp $1
git commit -m "Truncated history"
git rebase --onto temp $1 master
git branch -D temp

Requires Git 1.8+

Works well for mostly linear history (aka Git used for diff-ing of the catalog feeds).
